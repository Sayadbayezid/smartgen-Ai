## Commit Message Format

When using the automated commit script, outputs will be formatted following these strict examples:

**Example 1:**
Input: Added user authentication with JWT tokens
Output:
`feat(auth): implement JWT-based authentication`
`Add login endpoint and token validation middleware`

**Example 2:**
Input: Fixed bug where dates displayed incorrectly in reports
Output:
`fix(reports): correct date formatting in timezone conversion`
`Use UTC timestamps consistently across report generation`

**Rule Engine Standard:** 
Your generated scripts must follow this style: `type(scope): brief description`, followed by a detailed explanation.

# Blog Post Blueprint

## Overview
This blueprint scaffolds a new article specifically formatted for automated publishing workflows and static site generators like Hugo.

## Sequential Workflow
1. Unzip this generated `.skill` package.
2. Open the `templates/` folder to find your base markdown structure.
3. Fill in the required YAML frontmatter (Title, Date, Author, Categories, SEO Description).
4. Write the core content using standard Markdown syntax.
5. Move the completed file to your project's `content/posts/` directory (e.g., for your GenZ Frontier repository).
6. Push to GitHub to trigger your automated deployment pipeline.
